# basic-react-three-example

Ce projet est un exemple de base pour utiliser React et Three.js, grâce aux librairies [React Three Fiber](https://github.com/pmndrs/react-three-fiber) et [Drei](https://github.com/pmndrs/drei).
L'objectif de ce projet est de plus ou moins répliquer ce qu'on a fait précédemment en [WebGL](https://github.com/FYC-GROUP-9/basic-shader-example) et [three.js](https://github.com/FYC-GROUP-9/basic-threejs-shape).

## Installation

Pour installer le projet, il faut d'abord cloner le dépôt :

```bash
git clone https://github.com/FYC-GROUP-9/basic-react-three-example.git
```

Puis aller dans le dossier et installer les dépendances :

```bash
cd basic-react-three-example
npm install
```

## Lancement

Pour lancer le projet, il suffit de lancer la commande suivante :

```bash
npm run dev
```
